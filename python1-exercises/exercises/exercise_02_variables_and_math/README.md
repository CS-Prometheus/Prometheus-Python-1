# Exercise 02 — Variables and Math

Write a function called `rectangle_area` that takes `width` and `height`
and returns the area of the rectangle.

Then write a function called `is_even` that takes an integer and returns
`True` if it's even, `False` otherwise.

**Examples:**

```python
rectangle_area(3, 4)   # returns 12
is_even(6)              # returns True
is_even(7)              # returns False
```
