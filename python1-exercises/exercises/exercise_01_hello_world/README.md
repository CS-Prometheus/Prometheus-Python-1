# Exercise 01 — Hello World

Write a function called `greet` that takes a person's name as a string and
returns a greeting.

**Example:**

```python
greet("Ava")   # returns "Hello, Ava!"
greet("Sam")   # returns "Hello, Sam!"
```

Edit `exercise.py` and fill in the `greet` function. Then run the tests to
check your work.
